# 💰 Financial Calculator

A simple yet powerful console-based financial toolkit for everyday calculations.

## 🚀 Features

### ✨ Investment Calculator

- Simple interest
- Compound interest
- Quick estimation of profit over time

### 💳 Loan Calculator

- Monthly payment calculation
- Full repayment schedule

### 💱 Currency Converter

- Convert between currencies
- Track conversion history

### 📝 History Tracking

- Automatically saves results to JSON files
- Easy to review past calculations
